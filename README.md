# PMP Exam Study Tools  

Interactive, browser-based study tools for pmp exam preparation, highlighting evm understanding and ECO-PMBOK cross-referencing.
No installation required — works directly in any modern browser.

## Tools

- **[ECO → PMBOK Map](https://soriamoi.github.io/pmp-study-aids/)**
- **[EVM Study Session](https://soriamoi.github.io/pmp-study-aids/evm-study.html)**

## How to use

No login, no download, no setup. Open the links above in any browser. Works on desktop and mobile.
